# @fizmoo/core

The core library for fizmoo — types, `defineCommand`, `defineConfig`, `command()`, the build pipeline, and the runtime.

This package is consumed by the [`fizmoo`](../fizmoo) CLI. If you are building a CLI with fizmoo, install `fizmoo` rather than this package directly.

**Documentation:** [../docs](../docs/getting-started.md)
